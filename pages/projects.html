Automated Patch Backporting (Research)

LangGraph Spring Boot Python (MCP) TypeScript

A multi-agent, Large Language Model (LLM) system designed to automate the backporting of security patches in enterprise software. This research addresses the challenges of Semantic and Structural Divergence by using an orchestrated agentic workflow to reason about code changes, dependencies, and regressions.

Core Architecture

  • Note-Taking & Reasoning Agent: Analyzes patch semantics and maintains context.
  • Code Generation Agent: Synthesizes backported code variants.
  • Validation Agent: Enforces compilation and test integrity.

Impact & Methodology

  • Hybrid Detection: Combines Git history tracing with TF-IDF/Embeddings for location accuracy.
  • Validation: Tested on a curated dataset of 474 verified backports from OpenJDK and Kafka.
